# ng-error-tooltips demo app

This demo README intentionally stays short to avoid maintaining duplicated library documentation.

Use these READMEs as the canonical documentation:

- [Repository README](../README.md) for GitHub and general project documentation.
- [Package README](../ng-error-tooltips-lib/projects/ng-error-tooltips/README.md) for the npm package documentation that is built with the library.

Run the demo locally from this folder:

```bash
npm start
```

Build the GitHub Pages demo:

```bash
npm run deploy
```
